# Software Test Report - Amenities Button Issue

## Summary
The **Amenities** button on the website does not work. When clicked, nothing happens and no message is shown.

## Problem
- Button looks clickable but does nothing  
- No page opens, no content appears  
- No error message is shown  

## Expected Behavior
- Clicking the button should show amenities or open a related page  
- User should see a response or feedback  

## Test Cases

### 1. Check Visibility
- Open the website  
- Confirm the button is visible  

### 2. Click Test
- Click the button  
- Check if anything happens  
- Check console for errors  

### 3. Content Check
- Verify amenities content appears after clicking  

## Severity
- **High** - blocks access to an important feature  

## Suggested Fix
- Check button click event (JavaScript)  
- Check browser console for errors  
- Test on different browsers
